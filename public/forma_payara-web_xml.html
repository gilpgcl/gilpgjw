Lo necesitamos para que las formas envíen sus datos en "UTF-8" y puedan manejar correctamente acentos, así como otras letras que no se usan en el idioma inglés.

Para generar este archivo sigue los siguientes pasos:

  1. Selecciona el proyecto.
  2. Selecciona el Menú: File > New File…
  3. Aparece un cuadro de diálogo. Selecciona:
    Categories
    Payara
    File Types
    Payara Descriptor
    Click en Next >
  4. Clic en Finish
  5. Mover el archivo a la carpeta Web Pages/WEB-INF
