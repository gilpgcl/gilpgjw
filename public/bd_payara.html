Conexión a Bases de Datos en Payara

  1. Desde Netbeans selecciona la pestaña Services
  2. Abre la rama Servers
  3. Clic derecho en Payara Server
  4. Selecciona Start
  5. Cuando el servidor haya levantado, haz clic derecho otra vez en Payara Server
  6. Selecciona View Domain Admin Console
  7. Cuando abre la consola de Payara, en el árbol de la izquierda selecciona la rama Resources > JDBC > JDBC Connection Pools
  8. Clic en New…
  9. Introduce los siguientes datos:
    Pool Name
    pruebas
    Resource Type
    javax.sql.DataSource
    Database Driver Vendor
    MySql
    Clic en Next arriba a la derecha.
  10. En la página 2, el valor de Datasource Classname debe ser com.mysql.jdbc.jdbc2.optional.MysqlDataSource.
  11. Activa la opción Enabled de Ping.
  12. Avanza hasta Additional Properties y llena los siguientes datos, de acuerdo a la configuración de la base de datos.
    user
    pasausu
    databaseName
    pasabd
    password
    pasapass
    serverName
    localhost
    portNumber
    3306
    Clic en Finish abajo a la derecha.
  13. Para verificar la conexión, haz clic en el Pool Name llamado pruebas y cuando abra la pantalla, haz clic en Ping. Si todo está bien, debe indicar Ping succedded
  14. En el árbol de la izquierda selecciona la rama Resources > JDBC > JDBC Resources
  15. Clic en New…
  16. Introduce los siguientes datos de acuerdo con las configuraciones realizadas anteriormente:
    JNDI Name
    jdbc/pruebas
    Pool Name
    pruebas
    Clic en OK arriba a la derecha.
