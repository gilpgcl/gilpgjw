1. Desde Netbeans selecciona la pestaña Services
  2. Abre la rama Servers
  3. Clic derecho en Payara Server
  4. Selecciona Start
  5. Cuando el servidor haya levantado, haz clic derecho otra vez en Payara Server
  6. Selecciona View Domain Admin Console
  7. Cuando abre la consola de Payara, en el árbol de la izquierda selecciona la rama Configurations > erver-config > Security > Realms
  8. Clic en New…
  9. Introduce los siguientes datos:
    Name
    bd
    Class Name
    com.sun.enterprise.security.ee.auth.realm.jdbc.JDBCRealm
    JAAS Context
    jdbcRealm
    JNDI
    jdbc/pruebas
    User Table
    USUARIO
    User Name Column
    USU_CUE
    Password Column
    USU_MATCH
    Group Table
    USUARIO_ROL
    Group Table User Name
    USU_CUE
    Group Name Column
    ROL_ID
    Clic en OK arriba a la derecha.
  10. Reinicia el servidor.
